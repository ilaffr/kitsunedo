## 💡 Project Overview

**What:** Japanese/Korean language learning platform replacing fragmented 4-app stack

**Problem:** 
- Apps don't teach real proficiency (Duolingo: complete course but can't watch K-drama)
- Need 4+ apps at $14-24/mo (fragmentation)
- No grammar explanations (implicit fails for Asian languages)
- SRS burnout (WaniKani: 400-500 review avalanche)

**Solution:**
- Complete N5→N2 Japanese + TOPIK 1-4 Korean curriculum
- In-app grammar explanations
- Flexible SRS (user-controlled daily caps)
- JLPT/TOPIK mock exams (Premium)
- Real Content Reader (Premium)
- Ghost of Yotei aesthetic

**Business Model:**
- Free tier (80% value): Curriculum + SRS + Grammar
- Premium ($6.99/mo): Mock exams + Real content + Analytics
- Target: 15% conversion, break-even Month 10

**Market:**
- Japanese: $500M → $2.49B (16.43% CAGR)
- Korean: $7.2B → $67B (25.1% CAGR)
- N2 visa requirement (April 2026) creates urgency
